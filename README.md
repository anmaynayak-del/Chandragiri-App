# Chandragiri Farm Management App
A web-based farm management application used to manage worker attendance, wages, and loan calculations.
## Features
- Worker attendance management
- Wage calculation
- Loan tracking
- Responsive UI
- Cloud-based deployment
## Tech Stack
- HTML
- CSS
- JavaScript
- Firebase
- Cloudflare
